# Introdução ao Desenvolvimento Mobile

## Tipos de Desenvolvimento

### Nativo
- Android:
    - SDK : Android SDK
    - IDE : Android Studio
    - Linguagens: Kotlin e Java
    - Ambientes: Mac, Win, Linux

- IOS:
    - SDK: Cocoa Touch
    - IDE: Xcode
    - Liguagens: Swift / Objectype-C
    - Ambientes: Mac

### Multiplataforma
- React Native:
    - SDK: Node.js
    - IDE: VSCode
    - JavaScript / TypeScript
    - Ambientes: Mac, Win, Linux

- Flutter:
    - SDK: Flutter SDK
    - IDE: VSCode, Android Studio
    - Linguagens: Dart
    - Ambientes: Mac, Win, Linux
